# How I got reincarnated as a sorceror

Ceci est notre projet conçu durant les TP de réalité virtuelle.

Les ressources réalisées par nous-mêmes sont trouvables sur [ce dépôt](https://github.com/pgmtx/reincarnated-sorceror-assets/).
Les builds ainsi que le rapport sont trouvables dans le [release](https://github.com/BlueBerry1589/Reincarnated-sorceror/releases/tag/0.1).

## Logiciels utilisés

- Moteur de jeu : Unity v2022.3.37f1 ;
- Éditeur de code : JetBrains Rider ;
- Éditeur d'image : GIMP ;
- Éditeur audio : Audacity ;
- Gestionnaires de versions : git et [jujutsu](https://www.jj-vcs.dev/latest/).
